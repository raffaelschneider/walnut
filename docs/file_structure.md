# File Structure

Walnut uses a simple and intuitive directory structure to organize your knowledge base. Here's an overview:

```
~/Walnut/
├── Journal/
├── Notes/
│   ├── Personal/
│   ├── Projects/
│   └── Work/
├── References/
│   ├── Articles/
│   ├── Bibliography/
│   ├── Books/
│   ├── Contacts/
│   ├── Media/
│   ├── Papers/
│   ├── Podcasts/
│   └── WebContent/
├── Templates/
│   ├── Journal/
│   │   └── _record.md
│   ├── Notes/
│   │   ├── _note.md
│   │   ├── Personal/
│   │   ├── Projects/
│   │   └── Work/
│   └── References/
│       ├── Articles/
│       ├── Bibliography/
│       ├── Books/
│       ├── Contacts/
│       ├── Media/
│       ├── Papers/
│       ├── Podcasts/
│       └── WebContent/
└── walnut.yml
```
